# Tartarus

## Commits

### Commit format

*commit_type*(task):*commit message*

### Allowed commit types

| Commit type | Version increment |
| --- | --- |
| epic | major |
| major | major |
| feat | minor |
| build | patch |
| chore | patch |
| cicd | patch |
| docs | patch |
| fix | patch |
| perf | patch |
| refactor | patch |
| revert | patch |
| style | patch |
| test | patch |
